Ferramentas Utilizadas

Voltar

A principal ferramenta utilizada será o Visual Studio Code (VS Code). É necessário instalar a extensão Live Server para abrir o arquivo HTML no navegador. Utilizando o comando html:5 é possível criar a base de uma página em HTML.

Inspetor de Elementos

Voltar
O inspetor de elementos é um utilitário que permite visualizar o código-fonte subjacente de qualquer página web.1

Para acessar o inspetor de elementos é necessário apertar Fn+F12 no navegador (Google Chrome por exemplo). O arquivo CSS irá formatar o arquivo em HTML.

Estrutura Básica do HTML

Voltar

É uma convenção chamar a página principal de index em HTML. As tags HTML são utilizadas para informar ao navegador que tipo de estrutura é essa que está sendo construída, podendo ser títulos, parágrafos, imagens, links, entre outros. Sempre iniciam se com sinal de < e terminam com sinal de >, podendo haver uma / após o sinal de < no final indicando que a tag está sendo fechada.

Falando sobre Tags

Voltar

As tags principais para iniciar um HTML são:

  1. html
  2. head
  3. body
  4. i: coloca o texto em itálico
  5. strong: coloca o texto em negrito
  6. input: abre um campo para digitação
  7. img: adiciona uma imagem à página.

Atributos das Tags

Voltar

Atributos são propriedades que uma tag pode ter. Por exemplo:

  1. id: atributo global utilizado posteriormente com JavaScript, porém serve para identificar um determinado título por exemplo.
  2. style: atributo global utilizado posteriormente com CSS. Serve para mudar a cor das letras por exemplo.
  3. class: atributo global utilizado posteriormente com CSS. Cria uma categoria.
  4. i: atributo global que coloca o texto em itálico
  5. type: atributo específico da tag input que especifica qual tipo de caractere pode ser colocado no campo de digitação.

Textos

Voltar

Os h são títulos em HTML, temos h1, h2, h3 e assim por diante. O p define parágrafos. A tag blockquote cria uma citação. A tag u sublinha o texto. A tag mark funciona como um marca textos. A tag sup coloca o texto sobrescrito. A tag sub coloca o texto subscrito.

Listas Ordenadas e Não Ordenadas

Voltar

As listas permitem criarmos tópicos de textos para uma melhor exemplificação de um determinado assunto. A tag utilizada para criar listas ordenadas é ol. A tag li serve para colocar os itens da lista. A tag utilizada para criar listas não ordenadas é ul.

Links

Voltar

Quando clicamos em um link de uma pagina somos direcionados para outras paginas. A tag que cria link em HTML é a. Quando criamos o link desta maneira ele abre na mesma aba. Se quisermos que abra em uma aba diferente temos que utilizar o atributo target. Se o argumento for _self irá abrir na mesma aba, se o argumento for _blank irá abrir em outra aba. O atributo title cria um toolkit (balão) quando passa o mouse em cima do link.
